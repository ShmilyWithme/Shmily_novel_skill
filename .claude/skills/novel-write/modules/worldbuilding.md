# 世界观设定模块

## 功能说明

管理故事的世界观设定，包括背景、规则、势力等。

## 触发词

- `世界观`
- `设定`
- `背景设定`
- `规则体系`
- `设定世界观`

## 世界观设定模板

参考 `templates/worldbuilding-template.md`，核心结构：

```markdown
# 世界观设定

## 基础框架
- 世界名称：
- 时代背景：
- 科技/文明水平：
- 整体氛围/基调：

## 地理与环境
- 主要地域：
- 重要地点：（与剧情相关的场景）
- 特殊环境：

## 力量体系（如有）
- 体系名称：
- 等级划分：（从低到高，每个等级的能力描述）
- 突破条件：
- 特殊规则/限制：

## 社会结构
- 政治体系：
- 主要势力/组织：
- 经济体系：
- 社会阶层：

## 历史与文化
- 重大历史事件：
- 文化习俗：
- 信仰/宗教：

## 特殊设定
- 独特规则：（这个世界区别于现实的核心设定）
- 禁忌/禁区：

## 设定一致性追踪
| 设定项 | 出现章节 | 描述 | 状态 |
|--------|----------|------|------|
| ...    | ...      | ...  | ...  |
```

## 创建世界观流程

### 第一步：收集信息

询问用户以下信息：

1. **世界类型**：玄幻/科幻/历史/现代/架空
2. **时代背景**：古代/现代/未来/架空
3. **核心设定**：这个世界最独特的设定
4. **力量体系**：如有超自然力量，需要详细设定
5. **社会结构**：政治、经济、文化体系
6. **地理环境**：主要地域和重要地点

### 第二步：生成设定文件

根据用户提供的信息，生成 `worldbuilding/settings.md`

### 第三步：类型适配

根据选择的类型，参考对应的类型模板：
- `templates/genre-templates/xuanhuan-xianxia.md`
- `templates/genre-templates/urban-realistic.md`
- `templates/genre-templates/scifi-future.md`
- `templates/genre-templates/historical-fiction.md`
- `templates/genre-templates/mystery-thriller.md`

## 修改世界观流程

### 第一步：读取现有设定

1. 读取 `worldbuilding/settings.md`
2. 展示当前设定内容

### 第二步：询问修改内容

询问用户需要修改的部分：
- 基础框架
- 地理与环境
- 力量体系
- 社会结构
- 历史与文化
- 特殊设定

### 第三步：更新设定

根据用户要求更新世界观设定

## 设定一致性检查

### 检查内容

1. **力量体系一致性**
   - 等级划分是否清晰
   - 能力描述是否前后一致
   - 突破条件是否合理

2. **地理环境一致性**
   - 地点名称是否一致
   - 距离关系是否合理
   - 环境描写是否符合设定

3. **社会结构一致性**
   - 政治体系是否清晰
   - 势力关系是否合理
   - 经济体系是否自洽

4. **历史文化一致性**
   - 历史事件是否前后呼应
   - 文化习俗是否一致
   - 信仰体系是否完整

### 检查报告格式

```markdown
# 世界观一致性检查报告

## 检查范围
- 检查章节：ch001-ch020

## 检查结果

### 通过项
- [x] 力量体系描述一致
- [x] 地理环境描写符合设定
- [x] 社会结构前后呼应

### 问题项
- [ ] ch008: 力量等级描述与设定不符
- [ ] ch015: 地点距离与设定矛盾

## 建议修改
1. ch008: 将"斗皇"改为"斗宗"
2. ch015: 调整距离描述，或修改设定
```

## 设定追踪机制

### 追踪表维护

在 `worldbuilding/settings.md` 中维护设定一致性追踪表：

```markdown
## 设定一致性追踪
| 设定项 | 出现章节 | 描述 | 状态 |
|--------|----------|------|------|
| 力量体系 | ch001,ch005 | 练气-筑基-金丹 | 一致 |
| 地理设定 | ch003,ch010 | 东域-西荒-南疆 | 一致 |
```

### 追踪规则

1. 每次章节写完后，更新追踪表
2. 新出现的设定项要添加到追踪表
3. 发现矛盾时标记状态为"待修正"

## 注意事项

1. 世界观设定要详细，便于后续写作参考
2. 设定要自洽，避免前后矛盾
3. 力量体系要清晰，避免设定崩塌
4. 地理环境要合理，避免距离混乱
5. 社会结构要完整，避免逻辑漏洞
