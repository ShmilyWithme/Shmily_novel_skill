# 章节撰写模块

## 功能说明

根据大纲和上下文撰写章节正文。

## 触发词

- `写第N章`
- `生成第N章`
- `继续写`
- `下一章`

## 写前准备

每次写章节前**必须**执行以下步骤：

### 读取上下文

1. **P0 - 必读**
   - `outline/chapter-outs/chNNN.md`（本章梗概）
   - `style/style-config.md`（文风配置）

2. **P1 - 重要**
   - `chapters/ch[N-1].md`（前一章正文，用于衔接）
   - `chapters/ch[N-2].md`（前两章正文，用于语境，可选）

3. **P2 - 参考**
   - `characters/index.md`（角色索引）
   - 本章涉及的所有角色设定卡
   - `worldbuilding/settings.md`（相关世界观设定）

4. **P3 - 辅助**
   - `notes/misc.md`（伏笔追踪）

### 分析与检查

1. **分析**前文结尾，确认衔接点
2. **检查**本章需要埋设或回收的伏笔
3. **确认**本章涉及的角色和场景

### 禁止事项（强制）

1. **禁止重复内容**
   - 不得重复前一章的场景、对话、情节
   - 不得复制粘贴前文内容
   - 必须从新的时间点或场景开始

2. **必须读取前文**
   - 写章节前必须读取 `chapters/ch[N-1].md`
   - 分析前文结尾，找到自然衔接点
   - 确保时间线连贯，不跳跃

3. **检查重复**
   - 写完后自查：是否有与前章重复的对话
   - 自查：是否有与前章重复的场景描写
   - 自查：情节是否在原地踏步

## 正文生成规则

### 字数控制
- 默认字数：3000-5000字
- 允许浮动：±10%
- 根据 `project.md` 中的配置调整

### 文风要求
- 严格遵循 `style/style-config.md` 中的文风设定
- 保持人物言行与角色设定卡一致
- 章末必须有悬念钩子（除非是大结局章节）

### 结构要求
- 对话占比控制在 30%-50%（可配置）
- 动作描写与心理描写交替，避免大段独白
- 每个场景开头用环境/氛围描写过渡

### 章节结构
```
开头（环境/动作描写切入）
  ↓
发展（推进剧情，展开冲突）
  ↓
高潮（本章最精彩的部分）
  ↓
结尾（留下悬念钩子）
```

## 输出格式

生成的正文保存到 `chapters/chNNN.md`，格式如下：

```markdown
# 第N章 [章节标题]

[正文内容]

---
<!-- 元信息（不显示在正文中） -->
<!-- 字数: XXXX -->
<!-- POV: [角色名] -->
<!-- 场景数: N -->
<!-- 推进主线: [简述] -->
<!-- 埋设伏笔: [列表] -->
<!-- 回收伏笔: [列表] -->
```

## 写后处理

每次写完一章后，**必须**执行以下步骤：

### 1. 更新进度
更新 `project.md` 中的进度表：
- 将对应章节状态改为"已完成"
- 填入实际字数
- 填入完成日期

### 2. 更新伏笔（强制）
**必须**读取并更新 `notes/misc.md`：

**操作步骤：**
1. 读取当前 `notes/misc.md` 文件
2. 检查本章正文中的伏笔操作
3. 更新伏笔追踪表：
   - 新埋设的伏笔：添加新行，ID递增，状态设为"埋设中"
   - 已回收的伏笔：将对应行的状态改为"已回收"，填入实际回收章节
4. 如果有新的灵感或待办，更新对应区域

**示例格式：**
```markdown
| F007 | 新发现的伏笔内容 | ch009 | 第五卷 | — | 埋设中 |
```

### 3. 矛盾检查
如果发现前后矛盾，立即提醒用户

## 续写功能

### 触发词
- `续写第N章`
- `继续写`

### 执行流程
1. 读取第N章末尾内容
2. 分析当前情节状态
3. 继续延伸，不另起新章
4. 保持风格和情节连贯

## 改写功能

### 触发词
- `改写第N章`
- `重写第N章`

### 执行流程
1. 读取原有章节内容
2. 询问用户修改方向（如"加强打斗场面"、"对话更自然"）
3. 按照新方向重新生成章节
4. 保留原有元信息

## 扩写功能

### 触发词
- `扩写第N章`
- `扩展第N章第M段`

### 执行流程
1. 读取指定段落内容
2. 分析可以扩展的点
3. 进行扩展，保持原有风格
4. 更新字数统计

## 注意事项

1. 每次写作前必须读取上下文，不能凭空编造
2. 严格遵循文风配置，保持风格一致
3. 伏笔操作要记录在案
4. 字数控制在目标范围内
5. 章末必须有悬念钩子
