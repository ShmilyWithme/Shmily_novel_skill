# 网文写作助手 v2.0 更新公告

## 功能改进

**聊天式终端**
- 合并命令输入和回复为单一聊天输入框
- 用户消息显示 `>>>` 前缀，AI响应实时显示
- 终端输出实时刷新，无缓冲延迟

**日志面板**
- 新增独立"日志"标签页
- 实时记录命令执行、文件修改、错误信息
- 支持清空日志和导出为 .log 文件

**文件修改记录**
- 命令执行后自动显示新增/修改/删除的文件
- 记录到日志面板，方便追踪 AI 操作

**Skill 模块优化**
- 创建 templates/ 目录，包含5种类型模板
- 消除重复定义，统一改写/扩写功能
- 强化模块协作协议（角色变更传播、大纲偏离检测）
- 优化触发词，避免误触发
- 补充网文场景支持（支线管理、时间线、POV管理）

**其他改进**
- 大纲系统支持 `vol1-outline.md` 格式文件名
- 规划章节时自动读取伏笔笔记
- 写章节后强制更新伏笔状态

---

## 问题修复

- 修复 CTkLabel 不支持 weight 参数的错误
- 修复 stdin 警告提示
- 移除 emoji 图标，解决显示宽度不一致问题
