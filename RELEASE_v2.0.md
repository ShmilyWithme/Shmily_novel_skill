# 网文写作助手 v2.0 更新公告

## 全新 UI 设计

**浅色主题**
- 默认改为浅色模式，更清爽护眼
- 支持浅色/深色/系统三种主题切换

**统一色彩系统**
- 专业级色彩方案，视觉更舒适
- 输入框聚焦高亮，操作反馈更清晰
- Tabview 标签栏美化，选中态更明显

**界面优化**
- 所有文字颜色适配浅色/深色模式
- 按钮样式统一，交互体验更流畅
- 状态栏增强：显示项目路径、字数、时间

---

## 功能改进

**聊天式终端**
- 合并命令输入和回复为单一聊天输入框
- 用户消息显示 `>>>` 前缀，AI响应实时显示
- 终端输出实时刷新，无缓冲延迟

**日志面板**
- 新增独立"日志"标签页
- 实时记录命令执行、文件修改、错误信息
- 支持清空日志和导出为 .log 文件

**大纲系统**
- 修复卷大纲不显示的问题
- 支持 `vol1-outline.md` 格式文件名
- 大纲下拉菜单可选择"第1卷"、"第2卷"等

**伏笔追踪**
- 规划章节时自动读取伏笔笔记
- 写章节后强制更新伏笔状态
- 避免重复埋设或遗漏回收

**Skill 模块优化**
- 创建 templates/ 目录，包含5种类型模板
- 消除重复定义，统一改写/扩写功能
- 强化模块协作协议（角色变更传播、大纲偏离检测）
- 优化触发词，避免误触发
- 补充网文场景支持（支线管理、时间线、POV管理）
