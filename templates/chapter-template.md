# 章节正文模板

## 格式规范

```markdown
# 第X章 [章节标题]

[正文内容]

---
<!-- 元信息（不显示在正文中） -->
<!-- 字数: XXXX -->
<!-- POV: [角色名] -->
<!-- 场景数: N -->
<!-- 推进主线: [简述] -->
<!-- 埋设伏笔: [列表] -->
<!-- 回收伏笔: [列表] -->
```

## 正文写作要求

### 结构要求
1. **开头**：用环境描写或动作描写切入，快速进入情境
2. **发展**：推进剧情，展开冲突，深化人物
3. **高潮**：本章最精彩的部分（战斗、对决、揭秘等）
4. **结尾**：留下悬念钩子，吸引读者继续阅读

### 描写要求
- **环境描写**：每个场景开头用环境描写过渡
- **动作描写**：战斗场面要详细、有节奏感
- **心理描写**：展现人物内心世界，增加代入感
- **对话描写**：符合人物性格，推动剧情发展

### 节奏要求
- 对话占比：30%-50%
- 描写占比：30%-40%
- 叙事占比：20%-30%
- 避免大段独白或大段动作

### 字数要求
- 默认字数：3000-5000字
- 允许浮动：±10%
- 字数统计：以正文字数为准，不含元信息

## 章节类型

### 日常章节
- 推进人物关系
- 铺垫后续剧情
- 展现世界观细节
- 字数可适当减少

### 剧情章节
- 推进主线剧情
- 展开核心冲突
- 字数保持标准

### 高潮章节
- 战斗、对决、揭秘
- 节奏紧凑，张力十足
- 字数可适当增加

### 过渡章节
- 连接两个高潮
- 调整节奏
- 字数可适当减少

## 写后检查清单

- [ ] 是否遵循了章节梗概
- [ ] 是否与前文衔接自然
- [ ] 人物言行是否符合设定
- [ ] 是否埋设/回收了伏笔
- [ ] 章末是否有悬念钩子
- [ ] 字数是否在目标范围内
- [ ] 是否遵循了文风配置
