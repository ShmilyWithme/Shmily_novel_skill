# 文风参考指南

## 平台风格对比

### 起点风格
- **特点**：短句为主、节奏快、对话多、爽点密集
- **适合类型**：玄幻、都市、科幻
- **读者偏好**：喜欢快节奏、强爽感、升级流
- **写作要点**：
  - 每章都要有爽点
  - 对话要简洁有力
  - 动作描写要干脆利落
  - 避免大段心理描写

### 晋江风格
- **特点**：心理描写细腻、情感线优先、文笔优美
- **适合类型**：言情、古风、耽美
- **读者偏好**：喜欢细腻情感、优美文笔、深度人物
- **写作要点**：
  - 心理描写要细腻
  - 情感铺垫要充分
  - 文笔要优美流畅
  - 人物要有深度

### 番茄风格
- **特点**：通俗易懂、节奏极快、章节末必有钩子
- **适合类型**：都市、玄幻、悬疑
- **读者偏好**：喜欢轻松阅读、快节奏、强悬念
- **写作要点**：
  - 语言要通俗易懂
  - 节奏要快
  - 每章结尾必须有钩子
  - 避免复杂句式

### 文学风格
- **特点**：注重语言质量、叙事技巧、人物深度
- **适合类型**：现实、历史、文艺
- **读者偏好**：喜欢深度思考、优美语言、复杂人物
- **写作要点**：
  - 语言要精炼优美
  - 叙事要有技巧
  - 人物要有深度
  - 主题要有思考

## 文风参数详解

### 句式偏好
| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 短句为主 | 节奏快、冲击力强 | 动作场面、紧张时刻 |
| 长短交替 | 节奏丰富、张弛有度 | 通用场景 |
| 长句为主 | 节奏慢、描写细腻 | 心理描写、环境描写 |

### 描写密度
| 程度 | 特点 | 适用场景 |
|------|------|----------|
| 简洁 | 重点突出、节奏快 | 快节奏剧情、动作场面 |
| 适中 | 平衡叙事与描写 | 通用场景 |
| 细腻 | 细节丰富、代入感强 | 重要场景、情感高潮 |

### 对话风格
| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 口语化 | 自然亲切、代入感强 | 日常对话、轻松场景 |
| 文雅 | 有文化底蕴、格调高 | 古风、文艺场景 |
| 混搭 | 丰富多变、贴近人物 | 通用场景 |

### 幽默程度
| 程度 | 特点 | 适用场景 |
|------|------|----------|
| 无 | 严肃正经 | 严肃剧情、悲剧 |
| 轻度 | 偶尔轻松 | 调节节奏 |
| 中度 | 常有幽默 | 轻松向作品 |
| 重度 | 笑点密集 | 喜剧向作品 |

### 虐心程度
| 程度 | 特点 | 适用场景 |
|------|------|----------|
| 甜 | 轻松愉快 | 甜宠文、轻松向 |
| 微虐 | 小波折 | 增加戏剧性 |
| 虐 | 有虐有甜 | 虐恋情深 |
| 极虐 | 虐心虐身 | 悲剧向作品 |

### 节奏偏好
| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 快节奏 | 情节紧凑、进展快 | 爽文、升级流 |
| 张弛有度 | 有快有慢、节奏丰富 | 通用场景 |
| 慢热 | 慢慢展开、深度铺垫 | 文艺向、权谋向 |

### 信息密度
| 程度 | 特点 | 适用场景 |
|------|------|----------|
| 低 | 轻松易读 | 日常场景、过渡章节 |
| 中 | 适中信息量 | 通用场景 |
| 高 | 信息密集 | 重要剧情、设定展开 |

## 风格切换建议

### 切换时机
- 剧情转折点
- 人物状态变化
- 场景类型变化
- 读者反馈调整

### 切换注意事项
- 保持基本风格一致
- 渐进式切换，不要突变
- 记录切换原因
- 观察读者反馈

## 风格参考样本

### 起点风格样本
```
林轩眼中精光一闪，手中长剑猛然刺出。

剑光如电，直取对方咽喉。

"找死！"黑衣人冷笑一声，侧身躲过，反手一掌拍出。

林轩身形一转，避开掌风，剑势不减，反而更加凌厉。

"怎么可能？！"黑衣人脸色大变，他没想到这个年轻人的剑法竟然如此精妙。
```

### 晋江风格样本
```
月光如水，洒在青石板上，映出两道修长的身影。

沈清站在廊下，望着远处的人影，心中泛起一丝涟漪。

他想起三年前的那个夜晚，也是这样的月色，也是这样的离别。

"你......还好吗？"他轻声问道，声音里带着自己都未曾察觉的颤抖。
```

### 番茄风格样本
```
"什么？你要退婚？"

陈家大厅里，所有人都愣住了。

陈平站在大厅中央，嘴角勾起一抹冷笑。

三年前，他被未婚妻当众羞辱，被迫退婚。

三年后，他带着无上荣耀归来。

"没错，我要退婚。"陈平淡淡道，"不过这次，是我退你的婚。"
```
