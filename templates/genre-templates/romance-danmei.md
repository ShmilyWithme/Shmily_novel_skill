# 言情/耽美类型模板

## 类型特征

言情/耽美类网文以情感线为核心，主要特点：
- 以恋爱关系的发展为主线
- 人物情感和心理描写细腻
- 甜宠与虐心交替推进
- 人设鲜明，CP 感强烈
- 读者群体以女性为主

## 常见子类型

### 1. 现代言情
- 都市背景，职场/校园/家族
- 霸道总裁、天才医生、电竞选手等职业设定
- 强调现实感和代入感

### 2. 古代言情
- 宫廷/江湖/仙侠背景
- 宫斗、宅斗、江湖恩怨
- 权谋与爱情交织

### 3. 耽美（BL）
- 男性之间的恋爱故事
- 强强/弱强/年下等攻受设定
- 职业背景多样（电竞、娱乐圈、刑侦等）

### 4. 言情悬疑
- 以案件/事件推动感情线
- 推理与恋爱并行
- 适合喜欢烧脑+甜宠的读者

### 5. 重生/穿越言情
- 回到过去改变感情结局
- 穿越到异世界展开恋爱
- 信息差和先知优势带来的爽感

## 感情线设计

### 标准五阶段
1. **相识**：制造相遇契机，留下深刻第一印象
2. **暧昧**：日常互动增多，暗生情愫，试探对方
3. **确认**：表白或心意相通，关系确立
4. **考验**：外部阻力（家族反对、第三者、误会）或内心挣扎
5. **结局**：HE（幸福结局）或 BE（悲剧结局）

### 甜虐节奏
- **甜宠章**：撒糖、互动、日常、亲密接触
- **虐心章**：误会、分离、背叛、牺牲
- **建议比例**：甜 60% + 虐 30% + 过渡 10%
- **节奏控制**：3-5 章甜 → 1-2 章虐 → 1 章缓冲 → 循环

### 经典桥段
- 英雄救美 / 美救英雄
- 假结婚 / 契约关系
- 失忆 / 身份互换
- 日久生情 / 先婚后爱
- 破镜重圆 / 久别重逢
- 暗恋成真 / 双向暗恋

## 读者偏好

### 核心爽感
- **甜蜜互动**：男女主之间的化学反应
- **深情守护**：男主默默付出，女主感动
- **打脸情敌**：让第三者/反派自取其辱
- **误会解开**：虐后的甜更甜
- **圆满结局**：HE 是主流需求

### 雷点（务必避免）
- 男主渣到无法洗白
- 女主圣母/恋爱脑无底线
- 第三者戏份超过主角
- 结局强行 BE
- 感情线拖沓、反复拉扯

## 写作技巧

### 感情描写
- 用细节传达情感（眼神、小动作、下意识的关心）
- 内心独白要克制，少用"他/她想"
- 对话要有暧昧感，双关语和潜台词
- 环境描写烘托气氛（雨天、星空、花海）

### 节奏控制
- 甜宠章节奏可以慢，细细描写互动
- 虐心章节奏要快，不要拖泥带水
- 每章结尾留钩子（误会加深、心意暴露、危机出现）
- 高潮章节（表白、和好、大团圆）要给足篇幅

### 禁忌项
- 不要写"我配不上你"等老套台词
- 避免"全世界都反对我们"的狗血设定
- 不要让配角比主角更出彩
- 避免过度描写亲密场面（注意平台审核）
- 不要让误会持续超过 5 章（读者会弃文）
