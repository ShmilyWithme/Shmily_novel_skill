# 都市/现实类型模板

## 类型特征

都市/现实类网文以现代都市为背景，核心元素包括：
- 商战职场
- 情感纠葛
- 社会现实
- 人际关系
- 个人成长

## 常见子类型

### 1. 都市异能
- 主角获得特殊能力（透视、读心、系统等）
- 在现代社会中运用能力解决问题
- 保持现实感与超能力的平衡

### 2. 商战职场
- 从底层奋斗到商业帝国
- 商业策略和人际博弈
- 职场规则和潜规则

### 3. 都市情感
- 爱情、友情、亲情为主线
- 现实生活中的情感困境
- 人物心理的细腻描写

### 4. 重生/穿越
- 回到过去，利用先知优势
- 改变命运，弥补遗憾
- 信息差带来的爽感

## 升级体系

### 商战升级体系
- 小职员 → 主管 → 经理 → 总监 → 副总 → 总裁 → 商业帝国

### 财富升级体系
- 穷困 → 小康 → 中产 → 富裕 → 豪门 → 顶级富豪

### 影响力升级体系
- 无名小卒 → 业内新秀 → 行业领袖 → 商业巨头 → 传奇人物

## 读者偏好

### 核心爽感
- **打脸逆袭**：被人看不起后翻身
- **商业成功**：投资眼光精准，商业布局高明
- **情感收获**：获得美人芳心
- **社会地位提升**：从底层到上流社会

### 情感线节奏
- 相遇 → 相知 → 误会 → 和好 → 确认关系 → 考验 → 修成正果
- 支持多条情感线并行
- 配角情感线要与主线交织

## 现实逻辑约束

1. **商业逻辑要合理**：投资回报、商业模式要符合现实
2. **法律边界要清晰**：不能公然违法，要有灰色地带的描写
3. **社会规则要遵守**：权力运作、人际关系要符合现实
4. **情感发展要自然**：不能强行撮合，要有铺垫

## 章节结构参考

```
日常/铺垫（建立情境）
    ↓
冲突/机遇（出现转机）
    ↓
应对/行动（主角出手）
    ↓
结果/悬念（收获与新问题）
```

## 写作要点

### 对话技巧
- 对话要符合人物身份
- 商业谈判要有策略感
- 情感对话要有细腻感
- 日常对话要自然流畅

### 细节描写
- 场景描写要现代化
- 物品描写要符合时代
- 社会现象要贴近现实
- 细节要增加代入感

### 节奏控制
- 商战线要有张有弛
- 情感线要有起有伏
- 日常线要轻松有趣
- 高潮点要密集集中

## 禁忌事项

1. **避免逻辑漏洞**：商业操作要合理
2. **避免过度YY**：不能太脱离现实
3. **避免价值观问题**：不能宣扬负面价值观
4. **避免配角工具化**：配角要有自己的故事
5. **避免情感线混乱**：感情发展要有逻辑
