# 玄幻/仙侠类型模板

## 类型特征

玄幻/仙侠是网文最大的品类，核心元素包括：
- 修炼升级体系
- 宗门/门派系统
- 丹药法宝
- 秘境探险
- 天才对决

## 力量体系模板

### 标准仙侠体系
- 练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 大乘期 → 渡劫期

### 通用玄幻体系
- 斗者 → 斗师 → 大斗师 → 斗灵 → 斗王 → 斗皇 → 斗宗 → 斗尊 → 斗圣 → 斗帝

### 自定义体系设计要点
1. 等级数量：7-12级为宜
2. 每级要有明确的能力跃升
3. 突破条件要多样化（资源、悟性、机缘）
4. 设置瓶颈增加戏剧性

## 常见情节模式

1. **废柴逆袭**：主角从底层崛起，打脸看不起他的人
2. **天才陨落**：曾经的天才跌落谷底，重新崛起
3. **秘境探险**：进入秘境获取机缘和宝物
4. **宗门大比**：比武招亲、门派比试
5. **拍卖会**：争夺珍稀资源
6. **越级挑战**：以弱胜强，展现主角天赋
7. **势力争霸**：宗门、家族之间的争斗

## 读者偏好

### 爽点节奏
- **小爽点**：每3-5章一个（打脸、获得宝物、突破小境界）
- **大爽点**：每30-50章一个（突破大境界、击败强敌、身份揭露）

### 核心爽感
- 打脸：被人看不起后展现实力
- 捡漏：别人不识货，主角获得宝物
- 突破：关键时刻突破新境界
- 收服：强者臣服或加入

## 章节结构参考

```
开头钩子（悬念/冲突）
    ↓
冲突升级（敌人出现/危机降临）
    ↓
小高潮（战斗/对决/突破）
    ↓
留悬念（新危机/新线索）
```

## 禁忌事项

1. **避免过度水文**：每章必须有实质性进展
2. **避免主角无脑开挂**：胜利要有逻辑支撑
3. **避免等级碾压**：不要让主角一路碾压，要有挑战
4. **避免设定崩塌**：严格遵守已建立的力量体系
5. **避免配角脸谱化**：反派也要有动机和逻辑

## 写作要点

### 战斗描写
- 注重招式名称和视觉效果
- 加入心理博弈和策略
- 控制节奏，张弛有度
- 用环境描写烘托氛围

### 升级节奏
- 前期快速升级（吸引读者）
- 中期适当放慢（增加深度）
- 后期关键突破（高潮迭起）

### 世界观构建
- 地图要逐步展开
- 势力关系要清晰
- 历史背景要有深度
- 特色设定要鲜明
