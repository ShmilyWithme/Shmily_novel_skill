# 统计分析模块

## 功能说明

提供写作进度、字数统计、角色出场等数据分析。

## 触发词

- `进度`
- `字数统计`
- `角色出场统计`
- `节奏分析`
- `伏笔追踪`

## 统计功能

### 1. 写作进度统计

#### 触发词
- `进度`
- `写作进度`
- `完成情况`

#### 统计内容
- 总章节数 / 已完成章节数
- 总字数 / 目标字数
- 完成百分比
- 预计剩余时间

#### 输出格式

```markdown
# 写作进度报告

## 总体进度
- 总章节：XX章
- 已完成：XX章
- 进行中：XX章
- 待写：XX章

## 字数统计
- 总字数：XX万字
- 目标字数：XX万字
- 完成率：XX%

## 时间统计
- 开始日期：YYYY-MM-DD
- 平均每天：XX字
- 预计完成：YYYY-MM-DD

## 最近进度
| 章节 | 状态 | 字数 | 完成日期 |
|------|------|------|----------|
| ch020 | 已完成 | 3500 | 2024-01-20 |
| ch021 | 进行中 | - | - |
| ch022 | 待写 | - | - |
```

### 2. 字数统计

#### 触发词
- `字数统计`
- `统计字数`

#### 统计内容
- 各章字数
- 总字数
- 平均字数
- 最长/最短章节

#### 输出格式

```markdown
# 字数统计报告

## 总体统计
- 总字数：XX万字
- 章节数：XX章
- 平均字数：XX字/章
- 最长章节：ch015（XXXX字）
- 最短章节：ch003（XXXX字）

## 各章字数
| 章节 | 字数 | 偏差 |
|------|------|------|
| ch001 | 3500 | +500 |
| ch002 | 3200 | +200 |
| ch003 | 2800 | -200 |
| ... | ... | ... |

## 字数分布
- 3000字以下：X章
- 3000-4000字：X章
- 4000-5000字：X章
- 5000字以上：X章
```

### 3. 角色出场统计

#### 触发词
- `角色出场统计`
- `角色统计`

#### 统计内容
- 各角色出场次数
- 各角色出场章节
- 角色重要度排名

#### 输出格式

```markdown
# 角色出场统计报告

## 总体统计
- 总角色数：XX人
- 活跃角色：XX人
- 退场角色：XX人

## 角色出场排名
| 排名 | 角色名 | 出场次数 | 出场章节 | 类型 |
|------|--------|----------|----------|------|
| 1 | 主角 | 20 | ch001-ch020 | 主角 |
| 2 | 女主 | 18 | ch001-ch020 | 主角 |
| 3 | 反派 | 12 | ch005-ch020 | 配角 |
| ... | ... | ... | ... | ... |

## 角色出场频率
- 每章都出场：主角、女主
- 高频出场（>50%）：反派、配角A
- 中频出场（20-50%）：配角B、配角C
- 低频出场（<20%）：龙套A、龙套B

## 角色关系图
[可选：生成角色关系图]
```

### 4. 节奏分析

#### 触发词
- `节奏分析`
- `情节分析`

#### 分析内容
- 情节张力曲线
- 高潮分布
- 节奏变化

#### 输出格式

```markdown
# 节奏分析报告

## 情节张力曲线
```
张力
 │
 │    ╱╲      ╱╲
 │   ╱  ╲    ╱  ╲
 │  ╱    ╲  ╱    ╲
 │ ╱      ╲╱      ╲
 └─────────────────────→ 章节
   ch005  ch010  ch015  ch020
```

## 高潮分布
- 小高潮：ch005, ch010, ch015, ch020
- 大高潮：ch010, ch020

## 节奏评估
- 整体节奏：[快节奏/张弛有度/慢热]
- 高潮密度：[每X章一个小高潮]
- 张力变化：[平稳/波动/递增]

## 建议
1. ch008-ch010 节奏偏慢，建议增加冲突
2. ch015-ch018 高潮过于密集，建议适当放缓
```

- `伏笔状态`

#### 追踪内容
- 已埋设伏笔
- 已回收伏笔
- 待回收伏笔

#### 输出格式

```markdown
## 总体统计
- 总伏笔数：XX个
- 已埋设：XX个
- 已回收：XX个
- 待回收：XX个

## 伏笔列表

| F001 | 主角手臂印记 | ch003 | ch015-ch020 | 埋设中 |
| F002 | 老者的最后一句话 | ch008 | ch050 | 埋设中 |

- F001: 计划在 ch015-ch020 回收，当前已到 ch018

```

## 数据来源

### 文件读取
- `project.md` - 项目信息和进度
- `outline/chapter-outs/` - 章节梗概
- `chapters/` - 章节正文
- `characters/` - 角色设定
- `worldbuilding/settings.md` - 世界观设定
- `notes/misc.md` - 伏笔记录

### 统计方法
1. 读取所有相关文件
2. 提取关键信息
3. 进行数据计算
4. 生成统计报告

## 注意事项

1. 统计要准确，不能有误
2. 报告要清晰，便于阅读
3. 建议要具体，便于执行
4. 数据要及时更新
5. 可视化要直观
